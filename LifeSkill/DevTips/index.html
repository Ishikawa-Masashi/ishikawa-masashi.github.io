404Lab

и®°еҪ•дёҖдёӢејҖеҸ‘ж—¶з”ЁеҲ°зҡ„дёҖдәӣжҠҖе·§жҲ–иҖ…ејҖеҸ‘е·Ҙе…·еёёз”Ёзҡ„й…ҚзҪ®пјҢж–№дҫҝжҹҘйҳ…гҖӮ

NPMвҖӢ

жҸҗеҚҮдёӢиҪҪйҖҹеәҰвҖӢ

дҪҝз”Ё nrmвҖӢ

иҮӘеҠЁжөӢиҜ•еҗ„д»“еә“е»¶иҝҹпјҢе№¶еҸҜйҡҸж„ҸеҲҮжҚўд»“еә“гҖӮ

npm -g install nrm

жөӢиҜ•еҪ“еүҚд»ҘеҸҠе…¶д»–й•ңеғҸд»“еә“зҡ„е»¶иҝҹ:

D:\Code>nrm test

* npm ------ 836ms
  yarn ----- 813ms
  tencent -- 177ms
  cnpm ----- 1039ms
  taobao --- 234ms
  npmMirror - 14462ms

еҲҮжҚўй•ңеғҸд»“еә“:

D:\Code>nrm use cnpm  //switch registry to cnpm

Registry has been set to: http://r.cnpmjs.org/

и®ҫзҪ®д»ЈзҗҶвҖӢ

дҪҝз”Ё Clash дёәи®ҫзҪ®зі»з»ҹд»ЈзҗҶеҗҺпјҢеңЁ npm дёӯй…ҚзҪ®дҪҝз”Ёд»ЈзҗҶпјҡ

npm config set proxy=https://127.0.0.1:7890
npm config set https-proxy=https://127.0.0.1:7890

жҹҘзңӢй…ҚзҪ®жҳҜеҗҰжҲҗеҠҹпјҡ

npm config list

еҲ йҷӨд»ЈзҗҶй…ҚзҪ®пјҡ

npm config delete proxy
npm config delete https-proxy

жӣҝд»Је·Ҙе…·вҖӢ

PnpmвҖӢ

Pnpm:

pnpm install -g vite

YarnвҖӢ

Yarn:

yarn add -g vite

GitвҖӢ

и®ҫзҪ®д»ЈзҗҶвҖӢ

дҪҝз”Ё Clash и®ҫзҪ®зі»з»ҹд»ЈзҗҶеҗҺ,дёә Git и®ҫзҪ®д»ЈзҗҶпјҡ

git config вҖ”global http.proxy http://127.0.0.1:7890

git config вҖ”global https.proxy https://127.0.0.1:7890

и®ҫзҪ®й»ҳи®Өзј–иҫ‘еҷЁвҖӢ

жҲ‘д»¬йҡҸж—¶еҸҜд»Ҙжӣҙж”№ Git й»ҳи®Өзј–иҫ‘еҷЁ:

git config --global core.editor Code

дёҠйқўиҝҷиЎҢе‘Ҫд»Өзҡ„ж„ҸжҖқжҳҜ й…ҚзҪ®е…ЁеұҖ Git зј–иҫ‘еҷЁдёә VSCodeгҖӮ (Code жқҘиҮӘдәҺ VSCode зҡ„зҺҜеўғеҸҳйҮҸ D:\Microsoft VS Code\bin)

дҪҶжҳҜеҪ“дҪ жү§иЎҢ git commit ж—¶пјҢGit дјҡиҮӘеҠЁжү“ејҖ VSCode зј–иҫ‘еҷЁпјҢжҸҗзӨәдҪ иҫ“е…Ҙ commit зҡ„еҶ…е®№гҖӮ

дҪҶжҳҜпјҢGit е№¶дёҚдјҡзӯүеҫ…дҪ зҡ„иҫ“е…ҘиҖҢжҸҗзӨәжҠҘй”ҷдҝЎжҒҜ: Aborting commit due to empty commit message

дёәдәҶдҪҝ Git зӯүеҫ…дҪ зҡ„иҫ“е…ҘпјҢйңҖиҰҒеңЁеңЁжү§иЎҢж—¶ж·»еҠ  -w еҸӮж•°пјҢеҗҢж—¶дёәдәҶйҒҝе…ҚжҜҸж¬Ўж·»еҠ еҸӮж•°пјҢеҸҜд»ҘеңЁ config дёӯе…ЁеұҖй…ҚзҪ®пјҡ

git config --global core.editor Code -w

еҸӮиҖғпјҡhttps://stackoverflow.com/questions/9725160/aborting-commit-due-to-empty-commit-message
